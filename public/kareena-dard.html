About inline, internal and external

Inline CSS: Styles elements using style attribute.

Internal CSS: Styles written inside a <style> tag in the HTML head section.

External CSS: Styles placed in separate CSS files linked to the HTML page.

Embedded Style Sheets
Embedded style sheets are defined within the <style> tag in the HTML head. They're useful for styling a single page but not recommended for large projects.
